About Me

Currently studying at IIM in Paris, my goal is to specialize in video game development using my knowledge of multiple programming languages such as C#, Java and Python.
My current main project is GraphicsLabor, an Unity package aimed at helping game programmers and game designers to make games with Unity.

GraphicsLabor

Graphics labor is a unity package composed as of v4.0.0 of 4 parts:
- GraphicLaborer
- LaborerAttributes
- LaborerEditor
-LaborerTags
It is aimed at easing Hitboxes and HurtBoxes debuging for, mainly, fighting games as well as comming with Custom Attributes and a BitMask Tag system to help developing in general.
